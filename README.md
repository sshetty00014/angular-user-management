# Angular User Management App

## Features
- Users listing with API integration
- Manual search (frontend)
- Manual sorting (asc/desc)
- Manual pagination
- Dynamic routing
- Responsive UI
- Angular 18 standalone components

## Routes
- /users
- /users/:id
